          Cara Pasang Grafik Kunjungan
        

        

          

            Untuk menampilkan grafik kunjungan Real-Time, kita menggunakan
            Google Analytics + Looker Studio (Gratis). Ikuti langkah ini:
          

          
    
            
  1. 
              Daftar akun di
              Google Analytics
              dan pasang ID pengukurnya (G-XXXXX) di file
              index.html (bagian head).
            
    2. 
            
  2. 
              Buka
              Looker Studio.
            
    3. 
            
  3. Buat Laporan Baru (Blank Report).
    4. 
            
  4. 
              Pilih Sumber Data: Google Analytics -> Pilih Properti
              Website Kamu.
            
    5. 
            
  5. 
              Buat grafik (misal: Time Series Chart) yang menampilkan
              Views per tanggal.
            
    6. 
            
  6. Klik tombol Share -> Embed report.
    7. 
            
  7. Aktifkan "Enable embedding" dan pilih "Embed URL".
    8. 
            
  8. Copy URL Embed-nya.
    9. 
            
  9. 
              Di Panel Admin ini, klik tombol "Set Embed Code" di kotak
              grafik.
            
    10. 
            
  10. Paste URL tersebut. Selesai!
    11.
